# India Weather Dashboard

A fully coded weather dashboard focused on Indian regions with the following features:

1. Temperature and live key metrics.
2. Humidity tracking.
3. Rainfall map using OpenWeather tile layers.
4. Fog forecast for the next 24 hours.
5. Cyclone tracker alert UI for major coastal cities.
6. Latest weather news feed.
7. Built-in API/runtime error collection panel.

## Run locally
Open `index.html` in a browser, or serve this folder with any static server.

Example:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080/Weather%20Dashboard/`.
